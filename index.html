Steps to protect Your Computer from Cyber Threats

  • Install Internet Security Software
  • Install a Firewall
  • Create a Boot Disk
  • Backup Everything
  • Install Software Updates
  • Configure Strict Web Browser and Email Security Settings
  • Do not Install/Run Unknown Programs
  • Do not Open Unknown Email Attachments/Links
  • Disable Hidden Filename Extensions
  • Turn Off Computer and Disconnect from the Internet
